Crate bytes

用于操作内存中连续的一片数据.主要用于网络方面，但是其他方面的应用也是非常不错的.

bytes crate 提供了一个高效的字节缓存结构Bytes，和一些trait 用于实现缓冲区（Buf， BufMut).

Bytes

Bytes 数值通过允许多个Bytes对象指向相同的底层数据，来实现zero-copy网络编程.这些对象通过一个引用计数来控制，直到没有对象持有指向底层数据的指针，那底层数据就会被释放.

提供了一组方法来操作字节数据，先看一个官网上的例子：

首先，我们使用BytesMut申请了一块1024个字节（1K）大小的内存.然后像buf中写入一个二进制格式的字符串“hello word"， 再写入u16类型的数字1234.

然后调用buf.split()切除一个指向已经使用的内存的智能指针，那么这时buf中的内容应该是b"bello world\x04\xD2"因为u16类型的数字在内存中存储为16进制.1234 = 0x04D2.其中句柄a和b将共享底层缓冲区并维护将视图跟踪到由句柄表示的缓冲区中的索引.之后buf中的capacity() = 1024 - 11 - 2 - 13 = 998.

Buf， BufMut

这两个trait提供了对于buffers的读和写的操作方法.底层的数据存储有可能是一片连续的内存，但也有可能不是.例如：bytes是保证连续内存的缓冲区，但链表将字节存储在不相交的块中.Buf和BufMut维持了指向底层数组当前位置的指针，当数组读取或者写入时候，指针也会随之改变.

与 Read 和 Write 的关系

乍一看，Buf 和 BufMut 在功能上似乎与 std::io::Read 和 std::io::Write 重叠.但是，它们有不同的用途.缓冲区是作为参数提供给 Read::read 和 Write::write 的值.然后读取和写入可能会执行系统调用，这可能会失败.但是对 Buf 和 BufMut 的操作是绝对可靠的.

功能阅读

trait

Trait bytes::Buf

  • 从缓存中读取一个bytes数组

    缓冲区将字节存储在内存中，这样读取操作就不会出错.其底层的存储有可能不连续. 一个buf中存的是一个指向缓存的指针. 从 Buf 中读取会使指针位置随之前进. 它可以认为是一个高效的字节容器的迭代器. 最简单的buf就是一个&[u8]

需要实现方法

  • fn remaining(&self) -> usize

    返回当前剩余容量

  • fn chunk(&self) -> &[u8]

    返回一个切片,其开始位置位于当前指针位置,长度在0到Buf::remaining()之间.

  • fn advance(&mut self, cnt: usize)

    将指针向前移动cnt个字节后返回.

    如果cnt > self.remaining() 则会发生panic.

    如果cnt=self.remaning() ,则返回一个空数组的指针.

一些其他重要方法

  • fn reader() -> Reader where Self: Sized,

    创建一个定长的容器实现read方法

为外部类型实现trait

  • impl<T: Buf + ?Sized> Buf for &mut T

  • impl<T: Buf + ?Sized> Buf for Box

  • impl Buf for &[u8]

  • impl<T: AsRef<[u8]>> Buf for Cursor

  • impl Buf for VecDeqeue

内部实现

  • impl Buf for Bytes

  • impl<T, U> Buf for Chain<T, U> where T: Buf, U: Buf

一般用[u8]的数组和bytes进行相互转换是比较方便和常用的.

BufMut

和Buf的主要区别在于为bytes提供了一系列写的方法

需要实现的方法

  • fn remaining_mut(&self) -> usize

    返回一个当前可写的容量

    需要注意的是,remaining_mut返回的数值可能比真实的容量要小.

  • unsafe fn advance_mut(&mut self, cnt: usize)

    指针向前移动cnt个单位,这个函数是unsafe的,因为前进后的位置处有可能还没经过初始化.

    如果 cnt > self.remaining_mut()的话,会发生panic

  • fn chunk_mut(&mut self) -> &mut UnintSlice

    返回一个可变的slice.
